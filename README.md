# SQL Analytics & Data Exploration 
This repository is a comprehensive collection of high-performance SQL scripts designed for deep data exploration, advanced analytics, and automated reporting. These queries are optimized for data analysts and BI professionals looking to extract actionable insights from relational databases.

### Overview
The scripts in this repository provide a standardized framework for common analytical tasks. Whether you are performing ad-hoc discovery or building robust data models, these queries demonstrate best practices in SQL logic, performance optimization, and modular design.

## Key Analytical Themes
This library is organized into specific modules to streamline your workflow:

* **Database Exploration:** Scripts for schema discovery, identifying primary/foreign keys, and data profiling.
* **Measures & Metrics:** Templates for calculating KPIs, such as conversion rates, average order value, and churn.
* **Time-Based Trends:** Advanced time-series analysis including Year-over-Year (YoY) and Month-over-Month (MoM) growth.
* **Cumulative Analytics:** Running totals, moving averages, and cumulative distribution functions.
* **Segmentation:** Customer and product segmentation using techniques like RFM (Recency, Frequency, Monetary) analysis.
* **Cohort Analysis:** Tracking user behavior over time relative to their acquisition date.
